## Content

Module structure contains:
- Screen or screens
- Redux state
- Redux slice
- Components used only with this module

Example:

```
src
│
│
│
└───modules
│   │
│   │
│   │
│   └───companyOffers
│       │   CompanyOffers.tsx
│       │   companyOffersSlice.ts
│       │   companyOffersState.ts
│       │   companyOffersSlice.ts
│       └───components
│           │   CompanyOfferItem.tsx
│           │   CompanyOffersHeader.tsx
│           │
│           └───
```
