## Content

Set of regex functions.

### isEmail

Checks whether supplied string follows email format or not.

#### Params
| Name            | Type           | Required       | Default value  | Description
|:----------------|:---------------|:---------------|:---------------|:---------------------------    
| email           | `string`         | True           | -              | Email string

#### Usage example

```typescript jsx
isEmail("test@domain.com");
```
